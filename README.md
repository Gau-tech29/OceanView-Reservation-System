# OceanView Reservation System

Hotel management system for academic assignment demonstrating CRUD operations and session management.

## Features
- Staff & Guest Management
- Room Booking System
- Payment Processing
- Session-based Authentication

## Technologies
Java, JSP, Servlets, MySQL, Maven

## Setup
1. Import as Maven project
2. Create database `oceanview_db`
3. Update `DBConnection.java` with your DB credentials
4. Run on Tomcat 9+

Admin: admin / admin123
Staff: staff / staff123
